# Project Overview

PUNNARAJ organizes evidence, doctrine, architecture, and public explanation into durable layers.

The system is built for long-term recovery. It assumes that conversations, agents, vendors, and repositories may fail, so important context must be recoverable from reviewed files.

## Layers

- Zero Vault preserves private raw evidence.
- Beta Reference Layer provides public-readable working context.
- Project Atlas presents a clear public explanation.

This repository publishes the Atlas and Beta pages for `magga-113`.
