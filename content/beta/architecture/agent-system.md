# Agent System

The agent system exists to help preserve, review, structure, and publish PUNNARAJ knowledge without taking ownership of authority.

Agents should read the Beta Reference Layer as current working context and treat the Project Atlas as public presentation.

## Agent Responsibilities

- Improve durable repository structure.
- Preserve boundaries between raw, beta, and public layers.
- Add validation where publication risk exists.
- Keep changes small, reviewable, and reversible.
- Report files inspected, files changed, validation, risk, uncertainty, and next action.

Agents are participants in maintenance, not the source of truth.
