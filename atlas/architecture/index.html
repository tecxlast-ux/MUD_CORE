Public Architecture

PUNNARAJ separates knowledge into layers so each audience sees the right level of detail.

Public Model

  • Raw source material stays in the private evidence layer.
  • Reviewed beta references support active work.
  • The Project Atlas explains the system publicly.
  • Cloudflare provides routing and publication surface.
  • Vendors remain adapters, not sovereigns.

The architecture favors readable markdown, static output, and recoverable structure.
