Public Doctrine

PUNNARAJ is governed by a small set of public principles.

Principles

  • Truth before presentation.
  • Structure before scale.
  • Provenance is sacred.
  • Canonical records come before aliases.
  • Human authority remains clear.
  • Agents should improve recoverability, not create hidden dependency.

Doctrine is useful only when it is visible, reviewable, and bounded.
