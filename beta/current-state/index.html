PUNNARAJ Current State

PUNNARAJ is a future-proof knowledge substrate for preserving evidence, context, doctrine, architecture, and public explanation across human and agent work.

This repository is a publication surface, not the source of truth. It exposes reviewed and compressed reference material that can be read by humans and agents without exposing the Zero Vault.

Working Layers

  • Zero Vault stores raw and sensitive evidence outside the public site.
  • Beta Reference Layer contains reviewed working context for current coordination.
  • Project Atlas presents a clean public explanation for outsiders.
  • Cloudflare Pages project magga-113 is the current Control Plane and Project Atlas surface.

Operating Premise

Truth comes before presentation. Structure comes before scale. Provenance is sacred. Canonical records come before aliases. The system is designed for intergenerational continuity and mutual understanding between humans and AI agents.

Recovery Posture

Agents may fail. Vendors may change. Repositories may be interrupted. The system must remain recoverable through durable files, clear boundaries, and public-safe reference layers.
