Beta Doctrine Index

The doctrine layer records working rules for how PUNNARAJ preserves truth, protects boundaries, and keeps the system recoverable.

Core Doctrines

  • Truth discipline keeps facts separate from interpretation.
  • Visible rules make operating assumptions explicit.
  • Sovereign boundary protects human authority.
  • Failure recovery assumes agent and vendor failure.
  • Agent resilience makes context recoverable from files.
  • Vendor boundary treats platforms as adapters, not sovereigns.

These pages are beta references. They are public-readable working doctrine, not final canon.
