Agent Resilience

Agents may lose context, misunderstand intent, or overreach.

PUNNARAJ reduces those risks by giving agents durable reference layers with visible status and confidence metadata.

Agent Rules

  • Read the current reference layer before changing structure.
  • Preserve raw evidence boundaries.
  • Prefer additive structure over destructive rewrites.
  • State uncertainty when source material is compressed or beta.
  • Keep publication changes reviewable and reversible.

The system should not depend on any single agent remembering the past correctly.
